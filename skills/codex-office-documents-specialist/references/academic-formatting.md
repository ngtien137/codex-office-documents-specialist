# Academic Formatting

Use this reference for theses, dissertations, graduation reports, and formal academic reports when the user does not provide a different institutional template.

Source distilled from a local institutional thesis-formatting guide that was reviewed during skill development.

Treat these rules as institution-specific defaults. If the user gives another school or department guide, that guide overrides this file.

## Page Setup

- Paper size: A4 (`210 x 297 mm`)
- Orientation: portrait by default
- Margins:
  - top: `2 cm`
  - bottom: `2 cm`
  - left: `3 cm`
  - right: `2 cm`
- Header distance: `1.27 cm`
- Footer distance: `1.27 cm`

## Typography

- Encoding: Unicode
- Font: `Times New Roman`
- Main body size: `13 pt`
- Line spacing: `1.2 lines`
- First-line indent or tab stop: `1.25 cm`
- Paragraph spacing:
  - before: `6 pt`
  - after: `6 pt`

## Heading Rules

- Chapter title:
  - `14 pt`
  - bold
  - uppercase
  - centered
- Level 1 subsection:
  - `13 pt`
  - bold
  - uppercase
  - left aligned
- Level 2 subsection:
  - `13 pt`
  - bold
  - left aligned
- Level 3 subsection:
  - `13 pt`
  - italic
  - left aligned
- Do not place punctuation after chapter, section, or subsection titles.

## Page Numbering

- Do not print a page number on the cover page.
- Front matter before Chapter 1 uses lowercase Roman numerals:
  - `i, ii, iii, iv...`
- Main matter from Chapter 1 through references and appendices uses Arabic numerals:
  - `1, 2, 3...`
- Page number position: centered.
- Page number size: `13 pt`

## Structure Order

Typical order:

1. cover page
2. secondary cover page
3. acknowledgements
4. table of contents
5. abstract or summary if required
6. list of tables
7. list of figures
8. list of abbreviations if any
9. Chapter 1: introduction
10. Chapter 2: literature review
11. Chapter 3: research methods
12. Chapter 4: results and discussion
13. Chapter 5: conclusions and recommendations
14. references
15. appendices

## Content Styling

- Title page title:
  - `18 pt`
  - normal
  - centered
- Abstract:
  - `12 pt`
  - italic
  - justified
- Body text:
  - `13 pt`
  - normal
  - justified
- Scientific names:
  - `13 pt`
  - italic
  - justified
- Acknowledgements:
  - `13 pt`
  - normal
  - justified
- References body:
  - `13 pt`
  - normal
  - left aligned
- Equation indentation:
  - `2 cm`

## Section Numbering

- Number sections by chapter, for example:
  - `2.1`
  - `2.1.1`
- Do not exceed three numbering levels unless the user explicitly asks for a deeper hierarchy.
- Each numbered group should contain at least two sibling items.
  - Avoid a lone `2.1.1` without `2.1.2`.

## Tables

- Number tables by chapter:
  - `Table 1.1`
  - `Table 4.2`
- Table title:
  - above the table
  - `12 pt`
  - bold
  - left aligned
- Table body:
  - `12 pt`
  - normal
  - left aligned
- Table note or annotation:
  - below the table
  - `10 pt`
  - italic
  - left aligned
- If the table is taken from another source, cite it directly below the table.
- Common table border convention in the guide:
  - three strong horizontal rules are preferred
  - top rule heavier
  - header separator rule lighter
  - bottom rule heavier

## Figures

- Number figures by chapter:
  - `Figure 2.3`
  - `Figure 4.1`
- Figure title:
  - below the figure
  - `12 pt`
  - bold
  - centered
- Figure note or source:
  - `10 pt`
  - italic
  - right aligned
- If a figure is taken from another source, cite it directly below the figure.

## Wide Tables Or Figures

- If a table or figure must be landscape, rotate it so the top points toward the binding side.
- Re-check header, footer, and page number behavior on those pages after rotation.

## References

- Separate references by language when needed.
- Keep foreign-language references in the original language.
- Sort references alphabetically.
- For Vietnamese authors, keep the normal Vietnamese name order in the entry.
- If there is no author, sort by the issuing organization or publication title.
- For multi-author citations:
  - use `et al.` for English references
  - use `ctv.` for Vietnamese references when appropriate

## Practical Application In This Skill

When the user says a document is a thesis, dissertation, graduation report, or academic report:

- recommend these settings before editing
- convert fake heading formatting into real heading styles
- replace blank-line spacing with paragraph spacing or page breaks
- ensure front matter and main matter page numbering are distinct when the document uses sections
- keep tables and figures numbered by chapter
- update the table of contents after structural edits

If the user provides a school template or guide PDF, inspect that file and override this reference rather than mixing standards.
